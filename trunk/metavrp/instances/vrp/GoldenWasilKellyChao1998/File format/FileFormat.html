Description for Files of Breedam's

The format of data and solution files in all directories is as follows:

A) DATA FILES

The first line of the file has the structure :

n Q D d B

where

n  = umber of customers
Q  = maximum load of a vehicle
D  = maximum duration of a route
d  = service (or drop) time
B  = Best known value

After that, the depot coordinates (x,y) are shown. Finally, the next lines contain, for each customer, the following information:

x y q

where

x = x coordinate
y = y coordinate
q = demand
